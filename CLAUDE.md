# CLAUDE.md（access-vba-work）

## このリポジトリの目的
客先の Access VBA 開発を支援する。ここで生成したコードを push し、
客先のブラウザ（github.com）からコピーして Access VBE に貼り付ける運用。

## 絶対ルール
- **機密情報をコードに含めない**: 接続文字列・パスワード・社内サーバーパス・
  客先名や個人名が特定できる文字列は書かない。必要な場合はプレースホルダ
  （例: `<<サーバーパス>>`）にして、実値は客先で手入力する
- **1モジュール = 1ファイル**: src/ 配下に modXxx.bas / clsXxx.cls として保存
- **ファイル冒頭に貼り付け先を明記**: `' 貼り付け先: 標準モジュール modImport`
- **Option Explicit を必ず先頭に付ける**
- **コピペ単位で完結させる**: 1ファイルをそのまま全選択コピー→貼り付けで
  動く状態にする（分割貼り付けが必要な構成にしない）
- **VBA の制約に注意**: 64bit Office の場合は Declare PtrSafe、
  日本語変数名は避ける、行継続は ` _`
- **Access のバイナリ（.accdb / .laccdb / .mdb）は絶対にコミットしない**
- push は「上げて」と言われたら即実行してよい（コード自体が納品物ではなく
  転送手段のため、通常プロジェクトよりコミット許可のハードルは低い）
- 回答は常に日本語・丁寧語
