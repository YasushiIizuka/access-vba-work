# 現状まとめ

**最終更新日**: 2026-07-17

## 全体像
- 客先の Access VBA 開発を、GitHub private リポジトリ経由のコピペ運用で支援する環境
- 自宅・事務所で Claude Code がコードを生成して push → 客先ブラウザ（github.com）でコピー → Access VBE に貼り付け
- リポジトリ初期構築は完了（2026-07-17）。実際の VBA 開発タスクはまだ未着手

## 基本方針・判断軸
- 機密情報はコードに書かない（プレースホルダ方式）
- Access バイナリはコミットしない（テキストのみ管理）
- 1モジュール = 1ファイル、コピペ単位で完結させる
- PC 間の同期は GitHub の push / pull で行う（Dropbox 等の同期フォルダには置かない）
